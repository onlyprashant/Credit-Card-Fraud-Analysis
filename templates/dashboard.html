ℹ️ About Project

Credit Card Fraud Detection using Machine Learning

📌 Project Overview

This web application detects fraudulent credit card transactions using Machine Learning. It supports both manual transaction analysis and batch CSV analysis through a secure web interface.

📊 Dataset

Kaggle European Credit Card Dataset

  • ✔ 284,807 Transactions
  • ✔ 492 Fraud Cases
  • ✔ 30 Input Features
  • ✔ Highly Imbalanced Dataset

🤖 Models Compared

  • ✔ Logistic Regression
  • ✔ Random Forest
  • ✔ Extra Trees ⭐
  • ✔ XGBoost
  • ✔ LightGBM
  • ✔ CatBoost

🏆 Selected Model

Extra Trees Classifier


Selected after comparing multiple models based on Precision, Recall, F1-Score, ROC-AUC and PR-AUC.
